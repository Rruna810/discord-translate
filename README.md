# discord-translate

概要

本プロジェクトは、Discord上で動作するリアルタイム自動翻訳Botです。
ユーザーがコマンドを入力することなく、送信されたメッセージを自動で判定し、日本語 ⇄ 英語間の双方向翻訳を行います。

グローバルなコミュニティ環境において、自然な会話体験を損なわない設計を目指して開発しました。

✨ 主な機能

🌍 言語自動判定（日本語 / 英語）

🔁 双方向リアルタイム翻訳

💬 コマンド不要の自動処理

⚡ 軽量設計による安定稼働

🛠 拡張性を考慮した構造設計

🧠 技術構成

本Botは以下の技術スタックで構築されています。

Python

discord.py

GoogleTrans（Google翻訳APIラッパー）

Flask（ヘルスチェック用エンドポイント）

Render（クラウドホスティング）

UptimeRobot（定期Pingによる常時稼働維持）

無料ホスティング環境では一定時間アクセスが無い場合にスリープ状態へ移行するため、軽量Webサーバーを併設し、UptimeRobotによる定期リクエストでプロセスを維持する設計を採用しています。

🚀 デプロイ戦略

GitHubとRenderを連携させ、プッシュ時に自動デプロイが実行されるCI的運用を行っています。

無料環境における制限を回避するため、

Flaskで最小構成のWebサーバーを構築

外部監視サービス（UptimeRobot）から定期的にHTTPリクエストを送信

Botプロセスの停止を防止

という構成により、追加コストなしで安定した常時オンライン環境を実現しています。
